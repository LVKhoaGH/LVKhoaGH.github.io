1. Enterprise Network Design (CCNA Mega Lab)

🔍 Tình huống:  Thiết kế và mô phỏng hạ tầng mạng Campus cho 2 văn phòng chi nhánh.

⚙️ Hành động:  Triển khai kiến trúc mạng đa tầng. Cấu hình L2/L3 EtherChannel và giao thức OSPF để định tuyến. Đảm bảo tính sẵn sàng cao (HA) bằng HSRPv2. Tăng cường bảo mật Layer 2 với DHCP Snooping, DAI và Port Security. Quản trị thiết bị tập trung qua SSH, Syslog và NTP.

✅ Kết quả:  Hệ thống vận hành ổn định, tự động định tuyến lại khi đứt cáp (failover), mô phỏng thành công môi trường mạng chuẩn doanh nghiệp.

Tools: Cisco Packet Tracer  |  VLAN · Trunk · EtherChannel · HSRPv2 · OSPF · DHCP Snooping · DAI · Port Security

2. Fortigate Perimeter Security Implementation

🔍 Tình huống:  Triển khai hệ thống tường lửa bảo vệ mạng nội bộ và quản lý truy cập Internet.

⚙️ Hành động:  Cấu hình Source NAT (SNAT) cho người dùng ra Internet và Destination NAT (VIPs) để public máy chủ CCTV an toàn. Áp dụng các Security Profiles: Antivirus chặn file EICAR, Web Filter chặn tên miền không mong muốn, Application Control chặn BitTorrent và cấu hình DoS Protection chống Ping Flood. Thiết lập Traffic Shaping giới hạn và đảm bảo băng thông.

✅ Kết quả:  Kiểm soát thành công 100% lưu lượng in/outbound, bảo vệ mạng LAN khỏi các cuộc tấn công và cấp phát băng thông hiệu quả.

Tools: VMware · EVE-NG · FortiGate  |  SNAT · DNAT · VIP · DoS Protection · Web Filter · Antivirus · Traffic Shaping

3. System Administration & Infrastructure Monitoring

🔍 Tình huống:  Xây dựng và giám sát hệ thống máy chủ dịch vụ cốt lõi.

⚙️ Hành động:  Triển khai Windows Server 2016 làm Domain Controller (ADDS), tích hợp DNS, DHCP và phân quyền File Server qua NTFS. Xây dựng các dịch vụ mạng (FTP, Mail, NFS, Samba) trên Linux Rocky 9. Tích hợp hệ thống giám sát PRTG qua giao thức SNMP (Read-Only) để theo dõi băng thông các cổng kết nối theo thời gian thực.

✅ Kết quả:  Tối ưu hóa việc quản lý định danh người dùng tập trung và đảm bảo khả năng giám sát, phản ứng nhanh với các sự cố quá tải băng thông.

Tools: VMware · Windows Server 2016 · Linux Rocky 9 · PRTG  |  ADDS · DNS · DHCP · NTFS · vsftpd · Postfix · SNMP

4. Linux Open-Source Network Services (Rocky 9)

🔍 Tình huống:  Triển khai các dịch vụ mạng mã nguồn mở trên nền tảng Linux phục vụ vận hành nội bộ.

⚙️ Hành động:  Cài đặt và cấu hình toàn diện các dịch vụ cốt lõi trên Rocky Linux 9 bao gồm: SSH quản trị từ xa, Web/FTP Server (vsftpd), DNS Server (bind), Mail Server (Postfix/Dovecot/s-nail), hệ thống chia sẻ file (NFS, Samba), cấp phát IP (DHCP) và đồng bộ thời gian (NTP).

✅ Kết quả:  Xây dựng thành công hạ tầng dịch vụ mạng độc lập, cho phép client truy cập phân giải tên miền, gửi nhận mail và chia sẻ tài nguyên an toàn.

Tools: Linux Rocky 9 · OpenSSH · vsftpd · bind DNS · Postfix · Dovecot · NFS · Samba · DHCP · NTP

5. Infrastructure Monitoring with PRTG

🔍 Tình huống:  Xây dựng hệ thống giám sát tập trung để theo dõi tình trạng thiết bị mạng và lưu lượng băng thông.

⚙️ Hành động:  Khởi tạo PRTG Network Monitor. Thiết lập giao thức SNMP (Read-Only) trên các Router Cisco với community string bảo mật. Khai báo thiết bị (Local Probe) và cấu hình các Sensor chuyên dụng để quét toàn bộ dải cổng (Interface range).

✅ Kết quả:  Trực quan hóa dữ liệu mạng theo thời gian thực (Real-time). Phân biệt rõ trạng thái Up/Down của từng cổng vật lý và vẽ biểu đồ giám sát băng thông in/outbound chính xác.

Tools: PRTG Network Monitor · Cisco Router · SNMPv2/v3 · Bandwidth Sensors · Real-time Graphing

6. Cloud Networking & Virtualization (GCP)

🔍 Tình huống:  Triển khai hạ tầng máy chủ ảo và kiến trúc mạng trên môi trường điện toán đám mây Google Cloud Platform (GCP).

⚙️ Hành động:  Thiết kế mạng ảo VPC (Virtual Private Cloud). Cấu hình Firewall rules điều hướng traffic in/out an toàn. Khởi tạo và cấp dải IP cho các máy ảo (VM). Thiết lập kết nối VPC Peering để định tuyến giữa các mạng độc lập. Tích hợp công cụ Monitoring giám sát tải CPU của VM và cấu hình lưu trữ Log truy cập Web Server nội bộ.

✅ Kết quả:  Triển khai thành công hạ tầng mạng Cloud linh hoạt, bảo mật. Đảm bảo các VM ở các VPC khác nhau ping thông suốt và giám sát hiệu năng hệ thống chặt chẽ.

Tools: Google Cloud Platform (GCP) · VPC · VM Instances · VPC Peering · Cloud Firewall · Cloud Monitoring & Logging
