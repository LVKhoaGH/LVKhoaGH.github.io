System Overview

Là một Kỹ sư Mạng (Network Engineer) trẻ với nền tảng kiến thức vững chắc về quy hoạch hạ tầng (Routing/Switching), bảo mật hệ thống và quản trị máy chủ. Ưu tiên hàng đầu trong thiết kế mạng của tôi là tính Dự phòng (Redundancy), Bảo mật (Security) và Khả năng mở rộng (Scalability). Luôn sẵn sàng triển khai các giải pháp mạng thực tế trên thiết bị vật lý và môi trường Cloud.

Routing & Switching

  • Định tuyến: OSPF, Static Route, IPv6 Routing.
  • Chuyển mạch: VLAN, STP (Rapid PVST+), Layer-2/Layer-3 EtherChannel.
  • Dự phòng gateway: HSRP.
  • Wireless: Cấu hình WLC (Wireless LAN Controller).

Network Security

  • Fortigate: SNAT/DNAT, Firewall Policy.
  • Security Profiles: Web Filter, Antivirus, IPS, Traffic Shaping (chống DoS).
  • Layer 2 Security: Port Security, DHCP Snooping, DAI.
  • Access Control: ACLs, Local Auth.

System Administration

  • Windows Server 2016: ADDS (Domain/OU), DNS, DHCP, File Server, Remote Desktop.
  • Linux (Rocky 9): SSH, vsftpd, BIND, Postfix/Dovecot (Mail), Samba, NFS.

Cloud & Monitoring

  • Google Cloud Platform (GCP): Khởi tạo VPC, VPC Peering, Firewall Rules, Compute Engine.
  • Monitoring: PRTG Network Monitor (giám sát băng thông qua SNMP), GCP Cloud Monitoring.

Dự án & Lab tiêu biểu

Project Logs

1. Enterprise Network Architecture (CCNA Mega Lab)

Cisco IOS, OSPF, HSRP, Rapid PVST+, EtherChannel, IPv6
  • Thiết kế cấu trúc mạng Campus với các phân vùng VLAN riêng biệt cho PC, VoIP, Server và Wi-Fi.
  • Đảm bảo tính dự phòng (High Availability) bằng cách cấu hình HSRP cho Default Gateway và Layer-2/Layer-3 EtherChannel để gộp băng thông.
  • Tối ưu hóa Spanning Tree với Rapid PVST+ và bảo vệ hạ tầng Layer 2 bằng Port Security, DHCP Snooping, và Dynamic ARP Inspection (DAI).
  • Triển khai IPv6 Routing tĩnh và cấu hình cơ bản mạng không dây qua WLC.

2. Edge Security & Traffic Management

Fortigate Firewall (GUI), SNAT/DNAT, Security Profiles, Traffic Shaping
  • Cấu hình Firewall Policy để kiểm soát luồng truy cập; thiết lập SNAT cho người dùng LAN ra Internet và DNAT (Virtual IP) để publish dịch vụ nội bộ.
  • Áp dụng các lớp bảo mật: Chặn tải file chứa mã độc (kiểm thử với file EICAR) bằng Antivirus, chặn truy cập website không mong muốn bằng Web Filter.
  • Bảo vệ hệ thống khỏi tấn công từ chối dịch vụ (DoS Anomalies) và cấu hình Traffic Shaping để giới hạn băng thông (Max/Min Bandwidth) cho từng nhóm người dùng.
  • Thực hiện backup/restore cấu hình và cập nhật Firmware thiết bị an toàn.

3. System & Cloud Infrastructure Deployment

Windows Server 2016, Linux Rocky 9, Google Cloud Platform (GCP)
  • Windows Server: Xây dựng hệ thống quản trị tập trung ADDS (Domain lekhoa.local), phân quyền truy cập File Server qua Group Policy, cấp phát IP tự động (DHCP) và phân giải tên miền (DNS).
  • Linux: Cài đặt và vận hành các dịch vụ cốt lõi: FTP (vsftpd), Mail Server (postfix/dovecot), chia sẻ file đa nền tảng (Samba/NFS) và đồng bộ thời gian (NTP).
  • GCP: Thiết lập kiến trúc mạng Cloud với VPC, kết nối các VPC độc lập bằng VPC Peering và viết Firewall Rules bảo vệ các máy ảo (VM).

4. Network Monitoring & Alerting System

PRTG Network Monitor, SNMP, GCP Monitoring
  • Tích hợp PRTG giám sát thiết bị mạng vật lý (Cisco Router) qua giao thức SNMP (Read-Only).
  • Thiết lập các Sensor để theo dõi trạng thái Up/Down của Interface và đo lường băng thông Real-time.
  • Sử dụng GCP Monitoring tạo các Dashboard (Widget) theo dõi tài nguyên CPU của máy ảo và thiết lập Alerting Policy cảnh báo khi quá tải.
